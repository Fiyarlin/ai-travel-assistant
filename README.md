# Autonomous AI Travel Planning Assistant

AI-powered travel planner using n8n, OpenAI, and free APIs.

## Features
- Weather-aware itinerary planning
- Location search using OpenStreetMap
- Nearby hotel discovery (Overpass API)
- Dynamic UI generation
- Fully automated workflow

## Tech Stack
- n8n
- OpenAI
- OpenWeather API
- OpenStreetMap + Nominatim
- Overpass API

## How to Run
1. Import workflow JSON into n8n
2. Add API credentials
3. Execute workflow
4. Submit form to generate plan
